p, div, span태그 사용법

  • div(division) : 영역을 나누는 것 / 레이아웃,계층 나누기 용
  • p(paragraph) : 문단, 단락 나누는 용
  • span(span) : 적용범위
  • div > p > span 순서로 하위태그만 품을 수 있음
  • a, span, strong은 인라인요소로 p태그 하위요소로 들어감

스타일 입히는방법

  1. class를 이용한 스타일 주기, 여러가지 스타일 줄수 있음
  2. id를 이용한 스타일 주기, 한가지 스타일만 가질 수 있음, #이용
  3. 태그에 직접적으로 스타일 주는 방법, 모든 태그에 적용 됨
  4. 복합 선택자 :
  • 예) p.new-style : p태그에 있는 class가 new-style인 태그만
  • 예) div .new-style : div태그 아래 모든 자식들중 스타일이 new-style인 태그만 선택적용
  • 예) div>.new-style : div 바로 밑에 자식들중 스타일이 new-style인 태그만 선택
